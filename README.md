# 게시판
이 프로젝트는 Spring boot 와 JPA를 기반한 MYSQL을 사용해 게시글 CRUD, 게시판 정렬 및 해시태그검색 기능, 댓글 CRD, 인증기능을 구현했다.
## 개발 환경
IntelliJ IDEA
Spring Boot 3.3.6
(Spring 6.1)
Java 17
Gradle 8.11
## Dependencies
Spring Web
Lombok 1.18
Querydsl 5.1
Thyeleaf 3.1.2
Spring security 6.3.5
data rest 4.3.6
mysql-connector-j 8.3.0
H2 Database
Spring Boot DevTools
Spring Data JPA
